To run this java program, First of all import Base64 class file which is present in codecs library
that can be downloaded from below link:
http://commons.apache.org/proper/commons-codec/download_codec.cgi

To compile the program:
 javac Notepad.java
 java -cp . Notepad

Here "-cp ." is used to set classpath for current directory.

After that include the libraries and then complie java program.
